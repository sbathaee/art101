Lab 12: Conditionals

By Kenny Losier and Shayan Bathaee

Challenge

For this lab, our goal was to get comfortable using conditional statements in JavaScript. To do so, we were tasked with creating a program that sorts a person into a Hogwarts house when they input their name. This program relies heavily on if, else, and else if statements, as well as comparison operators.

Problems

In a general scope of the difficulty of this lab, we both did not encounter any problems at all with the regular tasks. The difficulties we encountered lied in Task X. We had a slight issue incorporating the house name in the same line, but soon found a span id that proved to be the exact solution to that problem. Another one of our problems with Task X was that of incorporating a description and name of each house in our results, which would replace the previous results, if there were any. There was plenty of repetition involved, but it was worth the extra work to implement the extra code.
